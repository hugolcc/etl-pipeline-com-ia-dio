# Pipeline ETL com IA em Python

Este projeto implementa um pipeline **ETL (Extract, Transform, Load)** utilizando Python e Inteligência Artificial.

## Tecnologias utilizadas

* Python
* Pandas
* OpenAI
* Google Colab

## Etapas do Pipeline

### Extract

Criação de dados fictícios de usuários.

### Transform

Geração de mensagens personalizadas utilizando Inteligência Artificial.

### Load

Salvamento dos dados em arquivo CSV.

## Arquivo gerado

usuarios_mensagens.csv
